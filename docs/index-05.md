## 练习20：if嵌套
完成以下任务：
1. 使用input()函数获取用户输入的年龄
2. 使用input()函数获取用户输入的身高
3. 使用嵌套的if结构判断：
   - 如果年龄大于等于18岁：
     - 如果身高大于等于1.7米，输出"可以参加篮球比赛"
     - 否则输出"身高不足，不能参加篮球比赛"

     - 如果身高大于等于1.6米，输出"可以参加青少年篮球比赛"
     - 否则输出"身高不足，不能参加青少年篮球比赛"
4. 使用格式化输出显示判断结果
